# Martian-Robots

## Requirements
- NodeJS >= v14
- Docker-Compose >= v1.13.0

## Getting Started

### Run locally
```bash
npm run docker:compose:up
```

```bash
npm install
```

```bash
npm run test
```

### Start developeing
```bash
npm run test:watch:debug
```

### Build image

```bash
npm run docker:build
```
